# Como usar | How to use
Como instalar: <br/>
wget https://github.com/fellipmelo/tunnelngrok.git <br/>
cd tunnelngrok <br/>
chmod 770 * <br/>
./tunnelngrok.py <br/>

para executar novamente basta digitar no terminal 'tn.py'

# Desinstalar

cd /etc/tunnelngrok/ <br/>
./uninstall.sh <br/> <br/>
